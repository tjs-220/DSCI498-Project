# DSCI498-Project

This project explores how diffusion‑based generative models can be adapted to perform effectively in low‑resource environments, where only a small number of training samples are available. Because diffusion models typically rely on large and diverse datasets, their behavior under extreme data scarcity remains an important practical question, especially for domains such as handwritten scripts, medical imaging, and other specialized visual categories. Using the Omniglot dataset—an ideal benchmark due to its naturally limited number of samples per character—the project investigates lightweight, reproducible strategies for few‑shot adaptation, including augmentation‑driven regularization, low‑rank fine‑tuning, and adjustments to classifier‑free guidance. The goal is to understand which techniques offer the best balance between sample quality, diversity, and training stability when data is severely limited. The final outcome includes a research‑style report, a reproducible experimental framework, and an open‑source implementation that together provide a clear, accessible evaluation of diffusion‑based image generation in constrained data settings, along with insights into the feasibility and limitations of these models for real‑world low‑resource applications.
